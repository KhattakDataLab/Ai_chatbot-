# AI Chatbot

An intelligent chatbot built with Streamlit and HuggingFace's Qwen 2.5 72B model.

## Features
- Real-time AI conversations
- Clean, modern dark theme UI
- Adjustable response length
- Chat history tracking

## Tech Stack
- **Frontend**: Streamlit
- **Model**: Qwen 2.5 72B Instruct
- **API**: HuggingFace Inference API

## Setup

1. Clone the repository
```bash
git clone https://github.com/KhattakDataLab/Ai_chatbot-.git
cd Ai_chatbot-
```

2. Install dependencies
```bash
pip install streamlit python-dotenv huggingface_hub
```

3. Create a `.env` file with your HuggingFace API token
```
HUGGINGFACEHUB_API_TOKEN=your_token_here
```

4. Run the app
```bash
streamlit run chatbot_ui.py
```
